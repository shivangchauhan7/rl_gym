# Lunar Lander RL Demo

AI agent learns to land a spacecraft using reinforcement learning (PPO algorithm).

## Install Dependencies

```bash
pip install flask numpy gymnasium box2d-py pygame opencv-python-headless torch
```

## Run Web App

```bash
python app.py
```

Open http://localhost:5000

## Watch Trained Agent

```bash
python play_trained.py
```

## Train New Model

```bash
python lunarpy.py
```
